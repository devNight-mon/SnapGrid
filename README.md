## 📸 Overview

SnapGrid is a high-performance, native Android image browsing app focused on:

- ⚡ Smooth scrolling with large datasets  
- 🧠 Efficient memory management  
- 🚀 Modern Android best practices  

> Designed to handle high-resolution images at scale without frame drops
